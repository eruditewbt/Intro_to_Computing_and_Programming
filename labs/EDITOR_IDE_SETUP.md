# Lab: Editor/IDE Setup (VS Code)

## Goal

Turn your editor into a “build cockpit”: run, debug, format, and use the terminal.

## Part 1: Open a Folder (Project-Based Work)

- Create a folder called `editor_lab`
- Open it in VS Code (“Open Folder”)

Proof:

- Screenshot showing the folder in the Explorer sidebar.

## Part 2: Integrated Terminal

Open VS Code terminal and run:

```bash
python --version
```

If `python` doesn’t work:

- Windows: try `py --version`
- Linux/macOS: try `python3 --version`

## Part 3: Run a Program

Create `hello.py`:

```python
print("Hello from VS Code")
```

Run it from the terminal.

## Part 4: Formatting

Enable “Format on Save”.

Proof:

- Screenshot of settings or short note: “format on save enabled”.

## Part 5: Debugging (Optional Early Intro)

- Set a breakpoint
- Step over one line

Proof:

- Screenshot of the debug panel.
