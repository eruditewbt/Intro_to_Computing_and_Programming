# Contributing

Contributions should make the course clearer, more teachable, or more usable.

Good contributions:

- improve explanations
- add diagrams
- improve setup instructions for real devices
- add beginner-safe labs
- fix broken commands or unclear steps
- improve project templates and proof-of-work flow

Contribution rules:

- keep instructions beginner-friendly
- prefer practical clarity over jargon
- include proof or reproduction steps for tooling changes
- do not add complexity unless it clearly improves learning
