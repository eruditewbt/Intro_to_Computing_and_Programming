# Start

If you are a learner, use this order:

1. Read `README.md`
2. Open `docs/index.html`
3. Open `docs/start.html`
4. Open `docs/setup.html` and set up your actual device
5. Run `labs/EDITOR_IDE_SETUP.md`
6. Run `labs/GIT_BASICS.md`
7. Join Discord and subscribe on YouTube
8. Start Week 1

Community links:

- Discord: https://discord.gg/8e4bQNknA
- YouTube: https://www.youtube.com/@eruditewbt
- Repo pathways: `COMMUNITY_AND_REPOS.md`
- Project setup guide: `PROJECT_PATHS.md`

If you are an instructor:

1. Read `INSTRUCTOR_GUIDE.md`
2. Read `MODULES.md`
3. Read `ASSESSMENT_RUBRICS.md`
4. Use `templates/LESSON_PLAN_TEMPLATE.md`
5. Post your weekly sprint using `templates/DISCORD_WEEKLY_POST.md`
6. Direct learners to the Pages site in `docs/`
