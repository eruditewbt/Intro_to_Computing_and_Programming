# Start

If you are new, use this order:

1. Read `README.md`
2. Read `SYLLABUS_WEEK_BY_WEEK.md`
3. Open `setup/SETUP_INDEX.md` and set up your actual device
4. Run `labs/EDITOR_IDE_SETUP.md`
5. Run `labs/GIT_BASICS.md`
6. Start Week 1

If you are an instructor:

1. Read `INSTRUCTOR_GUIDE.md`
2. Read `MODULES.md`
3. Read `ASSESSMENT_RUBRICS.md`
4. Use `templates/LESSON_PLAN_TEMPLATE.md`
5. Post your weekly sprint using `templates/DISCORD_WEEKLY_POST.md`
